# Flat Hunter
Автоматизированная микросервисная система непрерывного мониторинга сайтов аренда недвижимости
## Цель проекта
Система непрерывно мониторит площадки объявлений (первоначально - Яндекс.Недвижимость), фильтрует поступающие объекты по заданным критериям (Цена, Метраж, Локация) и мгновенно отправляет новые варианты пользователю в Telegram.
## Архитектура системы
Проект построен на базе микросервисной архитектуры и разворачивается в изолированных Docker-контейнерах:
* **Parser Service (`/parser`)** — скачивает HTML-страницы, парсит объявления и передает данные.
* **Bot Service (`/bot`)** — взаимодействует с пользователем через Telegram и отправляет уведомления.
* **Database (`PostgreSQL`)** — хранит историю объявлений, чтобы избегать дубликатов.
